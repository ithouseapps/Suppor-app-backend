# IT House Academy - Support App

Full-stack academic support management system for IT House Academy.

## Structure

- **backend/** - Django REST API
- **admin/** - Admin frontend (React + Vite)
- **support/** - Support staff frontend (React + Vite)
- **student/** - Student frontend (React + Vite)
- **frontend/** - Landing page (React + Vite)

## Tech Stack

- Backend: Python 3.12, Django 5.1, DRF, SimpleJWT
- Frontend: React 19, React Router 7, Vite 8, Tailwind CSS 4
- Database: SQLite3
- Notifications: Telegram Bot API, Browser Notification API
